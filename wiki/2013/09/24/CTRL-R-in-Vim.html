Paste from register in insert mode

CTRL-R is used to paste something while in insert mode, from any register.

  • CTRL-R % Current file name
  • CTRL-R * Clipboard
  • CTRL-R + Clipboard
  • CTRL-R / Last search pattern
  • CTRL-R : Last command

The most interesting is that it can evaluate a mathematical expression.

  • CTRL-R = Evaluate
